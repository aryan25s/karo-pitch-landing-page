## About the Project
This project is a demo landing page for **Karo Pitch**, a platform designed to help early stage founders from across the India to pitch their startups directly to investors.

The goal of the website is to clearly communicate the platform vision and encourage founders to apply for pitching opportunities.

---

## Features / Sections
- Hero Section
- About Karo Pitch
- How It Works
- Who Can Apply
- Investors Section
- Featured Startups
- About KaroStartup
- Final Call To Action

---

## Tech Stack
- HTML
- CSS
- JavaScript
- Parcel (for bundling)

## Purpose
This website was built as a **demo prototype assignment for KaroStartup** to demonstrate design thinking, user experience, and the use of AI assisted tools for rapid website creation.
